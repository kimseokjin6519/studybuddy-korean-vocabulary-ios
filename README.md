# StudyBuddy Korean Vocabulary

Korean vocabulary study app, built for learning Korean vocabulary from TOPIK / KIIP class textbooks

> App Store: https://apps.apple.com/kr/app/studybuddy-korean-vocabulary/id6755516696

---

## What it does

StudyBuddy helps users:

- review Korean vocabulary in a fast, simple flow
- practice recall through quiz-based repetition
- track progress across study sets
- stay consistent with a lightweight, mobile-first experience

The goal is simple: make it easier to come back every day and study without overthinking the process.

---

## Why I built it

I wanted a study app that felt:

- faster than a traditional flashcard tool
- simpler than a full classroom platform
- more focused on actual repetition and retention

StudyBuddy was built around the idea that language learning works better when the loop is tight:
see a word, recall it, move on, repeat.
